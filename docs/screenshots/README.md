## How to Add Screenshots

To add screenshots:

1. Run the KAIROS application
2. Take screenshots using Windows Snipping Tool (Win + Shift + S)
3. Save them in this directory with the filenames listed above
4. Recommended image size: 1366x768 or higher
5. Format: PNG with reasonable compression
